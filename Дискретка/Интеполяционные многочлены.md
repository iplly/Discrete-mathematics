Интерполяция - общее именование для техник восстановления функции по имеющимся набору значений.

$$P(x) = \sum_{i=0}^{n} y_i \cdot \ell_i(x), \quad 
\ell_i(x) = \prod_{\substack{j=0 \\ j \neq i}}^{n} \frac{x - x_j}{x_i - x_j}$$

**Интерполяционный многочлен Лагранжа** ^lagrange
Формула:$$P_n(x)=\sum^n_{i=0}y_i\cdot l_i(x),~l_i(x) =\prod_{\substack{j=0 \\ j \neq i}}^{n}\frac{x-x_j}{x_i-x_j}$$
Пример:
$(0,1),(1,3),(2,2)$
$l_0(x) = \frac{(x-1)(x-2)}{(0-1)(0-2)}$
$l_1(x) = \frac{(x-0)(x-2)}{(1-0)(1-2)}$
$l_2(x) = \frac{(x-0)(x-1)}{(2-0)(2-1)}$
$P_2(x) = 1\frac{(x-1)(x-2)}{(0-1)(0-2)}+3\frac{(x-0)(x-2)}{(1-0)(1-2)}+2\frac{(x-0)(x-1)}{(2-0)(2-1)} = -\frac{3}{2}x^2+\frac{7}{2}x+1$ 

**Интерполяционный многочлен Ньютона** ^5b85fe
Строится через разделённые разности. Форма для неравноотстоящих узлов:

$$P_n(x) = f[x_0] + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \dots$$

Разделённые разности:
$$f[x_i] = y_i$$
$$f[x_i,x_{i+1}] = \frac{f[x_{i+1}]-f[x_i]}{x_{i+1}-x_i}$$
$$f[x_i,x_{i+1},x_{i+2}] = \frac{f[x_{i+1},x_{i+2}]-f[x_i,x_{i+1}]}{x_{i+2}-x_i}$$

Преимущество: при добавлении нового узла не нужно пересчитывать все коэффициенты.
