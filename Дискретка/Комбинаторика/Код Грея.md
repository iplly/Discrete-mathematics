### Код Грея ^gray
Двоичный код, в котором соседние числа отличаются ровно в одном бите. Уменьшает ошибки при переключении (АЦП).

**Построение (рекурсивное):**
$$G_1 = [0, 1]$$
$$G_{n+1} = 0 \cdot G_n \parallel 1 \cdot \overline{G_n}$$
где $\overline{G_n}$ — отражённый $G_n$ (в обратном порядке).

**Пример:** $G_2 = [00, 01, 11, 10]$

**Из двоичного в код Грея:**
$$g_i = b_i \oplus b_{i+1}$$
(для младшего бита $b_{n+1} = 0$)

**Из кода Грея в двоичный:**
$$b_i = \bigoplus_{j=i}^{n} g_j$$
(накопление XOR, начиная со старшего бита)

### Задача об АЦП ^adc
Аналого-цифровой преобразователь:
1) Дискретизация аналогового сигнала по времени
2) Квантование по уровню
3) Кодирование отсчётов

Код Грея применяется для устранения ошибок на границах уровней квантования: при переходе между соседними значениями меняется только один бит.
