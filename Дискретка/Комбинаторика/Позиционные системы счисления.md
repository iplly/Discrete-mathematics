### Позиционные системы счисления ^positional_sys

**Постоянное основание:**
Число $N$ в системе с основанием $b$:
$$N = \sum_{i=0}^{k} d_i b^i, \quad 0 \leq d_i < b$$
Перевод из десятичной: последовательное деление на $b$, остатки — цифры от младшего разряда.

**Факториальная система (переменное основание):** ^factorial_sys
$$N = \sum_{i=1}^{k} a_i \cdot i!, \quad 0 \leq a_i \leq i$$
Применяется для перечисления перестановок (алгоритм Лемера).

**Алгоритм Лемера:** ^lehmer
По перестановке $\pi = (\pi_1, \dots, \pi_n)$ строится факториальное число:
- $a_i = \#\{j > i : \pi_j < \pi_i\}$ (количество меньших элементов справа)
- Номер перестановки: $N = \sum_{i=1}^{n} a_i \cdot (n-i)!$

Обратно: по номеру $N$ восстанавливается перестановка последовательным выбором.

Корректность доказывается индукцией по разрядам.
